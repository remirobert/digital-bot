# Digital Ocean bot

### Installation

Install first the needed packages:
```
npm install
```

Then create a **auth.json** file for configutation credential.

```json
{
  "verify_token":"2D7D1EE4",
  "facebook_token": "EAAN8kzgEXZA4BAAwv"
}
```
